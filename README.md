## 📌 Sobre a Formação
Este repositório contém o registro da minha jornada na **Formação Java Developer** da [DIO](https://www.dio.me/). O objetivo desta trilha é consolidar os pilares do desenvolvimento backend, desde a sintaxe fundamental até a construção de APIs robustas com **Spring Boot**.

Como graduada em ADS e atual estudante de **Engenharia de Software**, utilizo este espaço para aplicar boas práticas de desenvolvimento, padrões de projeto e organização de código.
